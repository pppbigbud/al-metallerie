# Images du Carrousel Hero

## Instructions

Placez vos images de carrousel dans ce dossier avec les noms suivants :

- `hero-1.jpg` - Première image du carrousel (Bienvenue)
- `hero-2.jpg` - Deuxième image (Créations sur mesure)
- `hero-3.jpg` - Troisième image (Formations)

## Recommandations

- **Format** : JPG ou PNG
- **Dimensions recommandées** : 1920x1080px (Full HD) ou 2560x1440px (2K)
- **Ratio** : 16:9
- **Poids** : Maximum 500KB par image (optimisez avec TinyPNG ou similaire)
- **Qualité** : 80-85% pour un bon compromis qualité/poids

## Images temporaires

En attendant vos vraies images, vous pouvez utiliser des images de placeholder :
- https://placehold.co/1920x1080/333/FFF?text=AL+Metallerie
- https://via.placeholder.com/1920x1080
